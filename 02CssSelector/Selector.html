De css selectors die ik ga onthouden

  1. object#id : om een specifiek object met een id aan te spreken
  2. object.class : om een object met een specifieke class aan te spreken
  3. object:nth-child(odd/even) : om een even of oneven child aan te spreken
  4. objectA + objectB : om object B aan te spreken die een buur moet zijn van object A
  5. [attribute] : om een alle elementen met een specifiek attribute aan te spreken
  6. element[attribute] : om specifieke elementen aan te spreken met specifieke attributen
  7. element[attribute="value"] : om specifieke elemeneten met een specifiek attribute met specifieke value aan te spreken
  8. [attribute*="value] : om alle attributen aan te spreken waarin de value voorkomt, zoals ter in water
  9. [attribute$=value] : idem maar dan einde van de attribute
  10. [attribute^=value] : idem maar het begin van de attribute
